# AI guardians - from code to Cloud

## Descripción General

Esta guía integral proporciona instrucciones paso a paso para implementar un pipeline seguro y moderno de desarrollo y despliegue de aplicaciones. La documentación abarca prácticas end-to-end desde el aprovisionamiento de infraestructura hasta la protección en tiempo de ejecución, aprovechando las herramientas líderes de la industria en seguridad y automatización.

## ¿Qué Aprenderás?

- **Despliegue de Infraestructura Basado en GitOps**: Implementar principios de infraestructura como código para desplegar y gestionar entornos de desarrollo desde cero utilizando metodologías GitOps.
- **Protección de Cargas de Trabajo en la Nube**: Obtener visibilidad y protección para tus aplicaciones utilizando **FortiCNAPP Cloud Workload Protection (CWP)**.
- **Desarrollo de Código Seguro**: Integrar seguridad en tus pipelines de CI/CD con escaneo y remediación automatizados mediante **FortiCNAPP Code Security**.
- **Protección de Aplicaciones en Tiempo de Ejecución**: Proteger aplicaciones contra amenazas en tiempo real con **FortiAppSec SaaS WAAP** (Web Application and API Protection).
